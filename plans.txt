




Laurie's ideas:
    every hex has a building. if you want empty hexes to exist, you simply create a new type of building that does nothing,
allowing for new 'empty' hexes to be added to a base.
    every hex on a base has an hp. once that hp reaches 0, the hex is reverted back to an unclaimed tile,
unless that hex is surrounded by base hexes with the same owner, then all buildings on that hex are destroyed
    'wall' tile does not exist. visually, there is a wall on the edges of the outer hexes on the base, but functionally they do nothing.
you can demolish buildings and get some of the resources back.

getNeighbors(): HexCoordinates  [] {
        const directions = [
            { q: 1, r: 0, s: -1 },  // right
            { q: 1, r: -1, s: 0 },   // top-right
            { q: 0, r: -1, s: 1 },   // top-left
            { q: -1, r: 0, s: 1 },   // left
            { q: -1, r: 1, s: 0 },   // bottom-left
            { q: 0, r: 1, s: -1 },   // bottom-right
        ];

        return directions.map(dir => ({
            q: this.coords.q + dir.q,
            r: this.coords.r + dir.r,
            s: this.coords.s + dir.s,
        }));
    }
  
    isTraversable(): boolean {
      return this.purpose === HexPurposeOptions.LAND;
    }
