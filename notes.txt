Laurie's ideas (optional):
- every hex has a building. if you want empty hexes to exist, you simply create a new type of building that does nothing,
allowing for new 'empty' hexes to be added to a base.
every hex on a base has an hp. once that hp reaches 0, the hex is reverted back to an unclaimed tile,
unless that hex is surrounded by base hexes with the same owner, then all buildings on that hex are destroyed
    
- 'wall' tile does not exist. visually, there is a wall on the edges of the outer hexes on the base, but functionally they do nothing.
you can demolish buildings and get some of the resources back.
    
- each building has an 'action'. When activated, it takes a certain amount of time to complete. e.g, a mine may take 5 seconds
to produce one iron. Some actions are automatic, e.g a mine produces one iron every 5 seconds without having to be activated,
but a barracks may need to be told what troop to produce. Some actions have a cost, e.g a barracks may require 5 iron to
make a certain troop.
    
- Palace tile produces some of each resource
Bases with a palace (home bases) have a small % reduction in the time taken to complete the action (the workers have a
moral boost when working so close to their king/queen).

- Biomes: areas on the board which specialise in a certain resource or have certain attributes. e.g a grassland biome may be
filled with more hexes with grassy terrain, and have a boost to food production (if such a resource exists).
the biome positions are randomised at the start of each game. Maybe prevent certain biomes from being next to one another
if there are balancing issues.

    getNeighbors(coords: HexCoordinates): HexCoordinates  [] {
        const directions = [
            { q: 1, r: 0, s: -1 },  // right
            { q: 1, r: -1, s: 0 },   // top-right
            { q: 0, r: -1, s: 1 },   // top-left
            { q: -1, r: 0, s: 1 },   // left
            { q: -1, r: 1, s: 0 },   // bottom-left
            { q: 0, r: 1, s: -1 },   // bottom-right
        ];

        return directions.map(dir => ({
            q: coords.q + dir.q,
            r: coords.r + dir.r,
            s: coords.s + dir.s,
        }));
    }
